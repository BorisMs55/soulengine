### SoulEngine

GUI Tool kit for PHP and DevelStudio.


#### Инструкции

Данная версия SoulEngine не совместима с последней версией
DevelStudio (3.0 beta2), убраны некоторые критические
части и кое-что изменено в SoulEngine модуле.


#### Сборка

1. Установить пакет php4delphi из папки проекта
2. Откройте файл проекта soulEngine.dpr в Delphi 2007/2006
3. Соберите проект, он будет находится в папке `build/`.

Если при запуске exe будет выдаваться ошибка, скорее
всего рядом нет php5ts.dll версии 5.3.8+ и более.


#### Дополнительные расширения

В папке `ext/` находятся исходники расширений, которые
используются в DevelStudio. Чтобы их собрать, необходимо
прописать путь к исходникам php4delphi в настройках вашей
среды или проекта.
